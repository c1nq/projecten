# Terminal Setup

Persoonlijke PowerShell 7 terminal setup met Oh My Posh, FastFetch en custom thema.

## Tools
- Oh My Posh - custom rood thema
- FastFetch - systeeminfo bij opstarten
- PSReadLine - autocomplete en syntax highlighting
- PowerShell 7.5.4

## Features
- Titelbar met batterij, IP, uptime en tijd
- C1NQ ASCII art bij opstarten
- Rode kleurenthema
- Glaseffect achtergrond
- SSH config voor servers
